# RW PhD Write 公理

## AXIOM-01：写作从章节功能和主张开始

- 规则：写作从章节功能和主张开始，不从漂亮句子开始。
- 来源原子：`RWS-PHD-WRITE-007`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：没有来源时，不生成事实性引用或伪造 DOI。

## AXIOM-02：每个事实性主张都要有用户材料或可核验来源

- 规则：每个事实性主张都要有用户材料或可核验来源。
- 来源原子：`RWS-PHD-WRITE-008`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：没有研究结果时，不写成研究已完成。

## AXIOM-03：来源支持范围小于句子范围时

- 规则：来源支持范围小于句子范围时，收窄句子。
- 来源原子：`RWS-PHD-WRITE-009`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：不为流畅删除限制、反例和不确定性。

## AXIOM-04：结果、解释、推断和建议要使用不同的证据强度

- 规则：结果、解释、推断和建议要使用不同的证据强度。
- 来源原子：`RWS-PHD-WRITE-010`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：不把 Skill 输出当成作者最终责任或期刊合规确认。

## AXIOM-05：Introduction 建立问题和缺口

- 规则：Introduction 建立问题和缺口，不能提前假定研究结论。
- 来源原子：`RWS-PHD-WRITE-011`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：没有来源时，不生成事实性引用或伪造 DOI。

## AXIOM-06：Methods 说明做了什么和为何这样做

- 规则：Methods 说明做了什么和为何这样做，不能用结果合理化方法。
- 来源原子：`RWS-PHD-WRITE-012`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：没有研究结果时，不写成研究已完成。

## AXIOM-07：Results 报告预定和实际分析

- 规则：Results 报告预定和实际分析，不用 Discussion 语言解释。
- 来源原子：`RWS-PHD-WRITE-013`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：不为流畅删除限制、反例和不确定性。

## AXIOM-08：Discussion 先回答研究问题

- 规则：Discussion 先回答研究问题，再比较证据、解释限制和外推边界。
- 来源原子：`RWS-PHD-WRITE-014`。
- 适用：根据用户提供的研究材料和可核验来源组织论文或 PhD 章节，不依赖个人本地语料。
- 停止：不把 Skill 输出当成作者最终责任或期刊合规确认。
