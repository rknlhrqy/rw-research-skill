# RW PhD Write 案例和反例

## 案例 1

- 输入：根据 10 篇已提供论文写一个 Discussion 小节。
- 必须做：确认章节类型、章节功能、读者、目标主张和允许使用的材料。；运行来源、因果强度、前后一致性、作者语气和修改范围检查。；章节、段落或论证骨架。
- 不能做：依赖未提供的本地工作区内容；生成不存在的论文、数据、DOI、工具结果或期刊要求；越过停止条件：没有来源时，不生成事实性引用或伪造 DOI。
- 下一步：rw-phd-tone。
