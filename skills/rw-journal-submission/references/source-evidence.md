# RW Journal Submission 来源证据

## 公开来源用途

- `icmje`：ICMJE Recommendations。作者责任、披露、投稿、重复发表和 AI 使用。 官方页面：https://www.icmje.org/recommendations/
- `credit`：CRediT Contributor Role Taxonomy。记录贡献角色；CRediT 不单独决定作者资格。 官方页面：https://credit.niso.org/
- `cope`：COPE Core Practices。处理作者、利益冲突、数据、同行评审和发表伦理。 官方页面：https://publicationethics.org/core-practices
- `equator`：EQUATOR Reporting Guidelines Library。按研究类型选择报告规范；报告规范不替代研究设计和偏倚评价。 官方页面：https://www.equator-network.org/reporting-guidelines/
- `orcid`：ORCID Public API。核验公开作者标识和公开记录。 官方页面：https://info.orcid.org/what-is-orcid/services/public-api/

## 边界

- 这里保存的是来源定位和短用途说明，不是完整标准副本。
- 报告规范不等于设计质量评价；偏倚工具不等于写作模板。
- 版本、政策、费用、API 和期刊要求在任务执行时核验。
