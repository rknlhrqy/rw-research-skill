# RW Journal Submission 公开标准

这些条目用于选择方法和核验当前版本，不替代原始标准。

## ICMJE Recommendations

- 官方页面：https://www.icmje.org/recommendations/
- 本 Skill 用途：作者责任、披露、投稿、重复发表和 AI 使用。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## CRediT Contributor Role Taxonomy

- 官方页面：https://credit.niso.org/
- 本 Skill 用途：记录贡献角色；CRediT 不单独决定作者资格。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## COPE Core Practices

- 官方页面：https://publicationethics.org/core-practices
- 本 Skill 用途：处理作者、利益冲突、数据、同行评审和发表伦理。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## EQUATOR Reporting Guidelines Library

- 官方页面：https://www.equator-network.org/reporting-guidelines/
- 本 Skill 用途：按研究类型选择报告规范；报告规范不替代研究设计和偏倚评价。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## ORCID Public API

- 官方页面：https://info.orcid.org/what-is-orcid/services/public-api/
- 本 Skill 用途：核验公开作者标识和公开记录。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。
