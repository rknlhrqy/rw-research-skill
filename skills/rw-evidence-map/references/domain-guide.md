# RW Evidence Map 选择表

| 情形 | 执行动作 | 不要做 |
|---|---|---|
| 结果口径一致 | 比较效应、精确性和偏倚 | 只数支持论文 |
| 结果口径不同 | 分簇并说明不可比较原因 | 强行统一 |
| 研究结论冲突 | 按设计、人群、测量和分析拆解 | 按期刊名气裁决 |
| 只有摘要 | 建立临时节点并标记证据范围 | 补写方法细节 |
| 准备写作 | 导出 claim-to-source 表 | 直接把图转成事实句 |

## 核心规则

- 证据图先固定节点定义，再建立关系。
- 研究结论和当前综合判断要分开保存。
- 不同构念、测量工具或时间点不能因为名称相近就合并。
- 效应方向、效应大小和不确定性要分开记录。
- 无统计学显著性不等于没有效应。
- 高偏倚风险研究可以保留，但不能与低风险研究等权处理。
- 冲突先检查人群、设计、测量、分析和时间差异。
- 证据缺口要区分没有研究、没有可用全文和结果未报告。
- 每条严格关系必须有原文位置或结构化记录支持。
- 证据确定性是结果层判断，不是给整篇论文贴一个分数。
- 图谱可以显示关系，不能自动完成因果推断。
- 新增证据要保留旧判断和变化原因。
