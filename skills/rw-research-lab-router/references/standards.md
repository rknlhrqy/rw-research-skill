# RW Research Lab Router 公开标准

这些条目用于选择方法和核验当前版本，不替代原始标准。

## NCBI Entrez Programming Utilities

- 官方页面：https://www.ncbi.nlm.nih.gov/books/NBK25501/
- 本 Skill 用途：检索和获取 PubMed 等 Entrez 数据库记录。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## Crossref REST API

- 官方页面：https://www.crossref.org/documentation/retrieve-metadata/rest-api/
- 本 Skill 用途：核验 DOI 和出版元数据；成员提交的元数据可能不完整。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## OpenAlex Developers

- 官方页面：https://developers.openalex.org/
- 本 Skill 用途：发现作品、作者、来源、机构和引用关系；使用前检查当前认证和额度。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。

## ORCID Public API

- 官方页面：https://info.orcid.org/what-is-orcid/services/public-api/
- 本 Skill 用途：核验公开作者标识和公开记录。
- 版本规则：涉及当前要求时重新打开官方页面，并在输出中记录核验日期。
